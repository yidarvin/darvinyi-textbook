Neural Networks

A neural network is a parameterized function — layers of linear transformations interleaved with nonlinearities. Understanding how information flows forward, and how gradients flow backward, is the foundation of everything that follows.

The Forward Pass

Each layer computes z = Wx + b, then applies a nonlinearity a = σ(z). Stack enough of these and the network can approximate any continuous function — the universal approximation theorem. The widget below lets you feel this directly: add hidden units and watch the approximation tighten.

Backpropagation

Training minimizes a loss L(ŷ, y) by computing gradients via the chain rule, propagated layer by layer from output back to input. Each weight update nudges parameters slightly downhill on the loss surface.

∂L / ∂W(ℓ)  =  δ(ℓ) · (a(ℓ−1))T
δ(ℓ)  =  ( W(ℓ+1)T δ(ℓ+1) ) ⊙ σ′(z(ℓ))

Activation Functions

Without a nonlinearity, stacking layers collapses to a single linear transformation — depth buys nothing. Activation functions break linearity and determine how gradients flow during training. ReLU dominates in practice due to its sparsity and gradient simplicity, but its variants address specific failure modes.
